## Project Overview

Expense Tracker Dashboard is a modern finance management web application developed using React and Vite. The application enables users to efficiently manage daily expenses through an interactive and responsive dashboard interface.

Users can add, edit, and delete expenses, categorize transactions, visualize expense distribution using charts, export reports as PDF files, and store data locally using browser LocalStorage.

The project focuses on frontend development, responsive UI design, state management, and data visualization.
